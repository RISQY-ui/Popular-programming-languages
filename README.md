# 🧠 Java vs Python vs JavaScript – Basic Syntax Comparison

A personal side-by-side comparison of fundamental concepts in three programming languages: **Java**, **Python**, and **JavaScript**.

Perfect for:
- Beginners who want to see syntax differences at a glance
- Developers transitioning between languages
- My own quick reference 😊

---

## 📊 Quick Comparison Table

| Concept       | Java                              | Python                | JavaScript                         |
|---------------|-----------------------------------|-----------------------|------------------------------------|
| **Variable**  | `int x = 5;`                      | `x = 5`               | `let x = 5;`                       |
| **Looping**   | `for(int i=0; i<5; i++) {...}`    | `for i in range(5):`  | `for(let i=0; i<5; i++) {...}`     |
| **If-else**   | `if (x > 0) {...} else {...}`     | `if x > 0:`           | `if (x > 0) {...} else {...}`      |

---

## 📁 How to Use

1. Clone this repository
2. Explore each language folder
3. Run the examples using:
   - Java: `javac File.java && java File`
   - Python: `python file.py`
   - JavaScript: `node file.js`
